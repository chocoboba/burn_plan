burn-plan
=========

Simulation and analysis of a personal financial portfolio

## Installation

Add this line to your application's Gemfile:

    gem 'burn_plan'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install burn_plan

## Usage

# in burn_plan folder, run
ruby -Ilib bin/burnit.rb
